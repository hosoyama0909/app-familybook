# SWE.1 ソフトウェア要求分析

システム要求（SYR）を、実装可能なソフトウェア要求（SWR）へ詳細化する。
ここでは Stop 1 で追加した **きろく機能** と **お子さん登録** を中心に詳細化する
（既存機能は SWR-CORE-* として概略のみ）。

## きろく機能（グズりボタン）— SYR-10, 11, 12 由来

| ID | ソフトウェア要求 | 由来(SYR) |
|----|------------------|-----------|
| SWR-LOG-01 | 6種のイベントボタン（😭グズり/🤱抱っこ/🍼ミルク/💩おむつ/😴ねんね/⭐神スポット）を表示する | SYR-10 |
| SWR-LOG-02 | ボタン押下で、その時刻(ts)を持つ記録を選択中のおでかけに追加する | SYR-10,11 |
| SWR-LOG-03 | 位置情報は許可時のみ非同期で取得し記録に付与する。未許可/失敗/非対応でも記録は成立する | SYR-11, N3 |
| SWR-LOG-04 | 押下前に入力した「ひとことメモ」を記録に添付し、入力欄はクリアする | SYR-10 |
| SWR-LOG-05 | 当日分のイベント種別ごとの件数サマリーを表示する | SYR-12 |
| SWR-LOG-06 | 記録を日付ごとにグループ化し、各日は新しい順に時系列表示する | SYR-12 |
| SWR-LOG-07 | 位置付き記録には外部地図（座標）へのナビリンクを表示する | SYR-03,11 |
| SWR-LOG-08 | 各記録を個別に削除できる | SYR-10 |
| SWR-LOG-09 | 各記録の時刻をあとから変更できる（当日に押せなかった分の補正）。変更後は時系列を再整列 | SYR-10 |
| SWR-LOG-10 | 各記録のメモをあとから編集できる | SYR-10 |
| SWR-LOG-11 | 各記録を「だれの記録か」（お子さん）に任意で紐づけられ、未指定（なし）にも戻せる。記録時にも選択可 | SYR-10,13 |
| SWR-LOG-12 | だれの記録かを 物語プロンプト・ふりかえり・冊子 に反映する（誰がグズった等） | SYR-16 |

## お子さん登録 — SYR-13 由来

| ID | ソフトウェア要求 | 由来(SYR) |
|----|------------------|-----------|
| SWR-CHILD-01 | ニックネームと生年月(YYYY-MM)でお子さんを登録できる | SYR-13 |
| SWR-CHILD-02 | 生年月から現在の年齢（X歳Yヶ月）を自動算出して表示する | SYR-13 |

## AI物語（Stop 3）— SYR-16, N7〜N9 由来

| ID | ソフトウェア要求 | 由来(SYR) |
|----|------------------|-----------|
| SWR-STORY-01 | ふりかえりに「物語を作る」ボタンを表示し、押下で中継Workerへ生成要求を送る | SYR-16 |
| SWR-STORY-02 | 送信ペイロードは最小化する（trip概要・子のニックネーム/年齢・イベント集計・予定+記録の時系列/メモ）。**lat/lng・写真・本名を含めない** | SYR-N7 |
| SWR-STORY-03 | 送信前に同意文（何を送るか）を表示する | SYR-N7 |
| SWR-STORY-04 | 生成結果を `trip.story={text,ts}` に保存し、再表示・作り直しができる | SYR-16 |
| SWR-STORY-05 | 中継Worker URL(`storyUrl`)を設定・保存でき、未設定時は設定導線を出す | SYR-N8 |
| SWR-STORY-06 | 通信失敗・未設定・オフライン時はトースト通知し、既存物語と他機能を保持する | SYR-N9 |
| SWR-WORKER-01 | Worker は GEMINI_API_KEY を secret 保持し、CORS付きで Gemini を代理呼び出しする | SYR-N8 |
| SWR-AWARD-01 | ふりかえりに、きろくから集計した「あるある大賞」（MVP・大忙し時間帯・件数）を表示する（AI不要） | SYR-16 |
| SWR-WORKER-02 | Worker は payload.prompt を優先して受け取り、汎用リレーとして動く（物語/手紙で共用・Worker変更不要） | SYR-16,17 |
| SWR-LETTER-01 | ふりかえりで、登録済みお子さんごとに手紙をAI生成し `trip.letters[childId]` に保存・再表示する | SYR-17 |
| SWR-LETTER-02 | 手紙プロンプトはアプリ側で組み立てる（子の呼び名・当時の年齢・その旅の記録から） | SYR-17 |
| SWR-HP-01 | ふりかえりで、そのおでかけの記録から親のがんばり度メーターを表示する（AI不要） | SYR-18 |
| SWR-LEARN-01 | ホームで、全記録から「グズり時間帯」「外出→初グズりの平均経過」「休憩の目安」を提示する（AI不要・記録3件以上） | SYR-19 |
| SWR-OPEN-01 | ちずで、地方別訪問数から新規開拓の提案文を表示する（AI不要） | SYR-21 |
| SWR-BOOK-01 | ふりかえりから、表紙・物語・手紙・記録・写真を組んだ冊子を生成し `window.print()` で印刷/PDF保存する（@media print） | SYR-22 |
| SWR-BOOK-02 | 冊子はエディトリアル装丁（表紙パネル・章番号・ドロップキャップ・レターペーパー・写真フレーム）とし、色は print-color-adjust で印刷に反映する | SYR-22 |

## 受け入れ基準（抜粋）
- SWR-LOG-03：位置情報をブラウザで拒否しても、記録は0件にならず時刻付きで残ること。
- SWR-CHILD-02：2021-05 生まれを 2026-07 時点で表示すると「5歳2ヶ月」であること。
